
# Basic NLP Practice

This repository demonstrates basic Natural Language Processing (NLP) concepts using Python.
It is created for learning and portfolio purposes.

## Features
- Text preprocessing (cleaning, lowercasing, removing special characters)
- Tokenization
- Stopword removal
- Stemming and lemmatization
- TF-IDF vectorization
- Sentiment analysis

## Technologies Used
- Python
- NLTK
- scikit-learn
- TextBlob
